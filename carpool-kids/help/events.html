Events

From the main menu, tap on "Events", then tap on the "+" button in the upper right of the Events list to create a new Event.

Creating an event is broken into 3 parts. The "Event" tab is where you specify the Date, Location and Name of the event, overriding the Carpool defaults if necessary. You can also add a note to be included with the event.

Events can also be created as "Repeating" by specifying a week repeat value (1 for every week, 2 for every other week, etc.) and which days of the week to repeat on.

Use the "To" tab to specify the starting time of the event and remove any of the kids as riders to the event if necessary. To remove a rider, swipe to the left on the rider's name to expose the "Not Riding" button.

You can also assign a Driver for the "To" leg at this time.

Use the "From" tab to set the ending time of the event and remove any riders as necessary.

You can also assign a Driver for the "From" leg at this time.
