Carpools

A carpool serves as a template for creating events. The carpool setup provides the default values for "who, what and where", while the events you create for the carpool define the "when" and allow you to override the carpool defaults. Events have both "To" and "From" driving legs by default, so it's not necessary to create separate "To" and "From" carpools!

Use the "+" button at the top right of the Carpools list to create a new Carpool.

Provide values for the carpool Name, Location, and Event name. These will be used as the defaults for new events created for the carpool.

Tap the "Add Driver" button to add one of your friends as a Driver for the Carpool.

Tap the "Riders(0)" tab to switch over to the Riders list. Use the "Add Rider" button to assign one of your kids or the kid of another Driver to the Carpool.

Select a kid to assign to the Carpool.

Once you've assigned Riders, tap the "Save" button on the top right.
