Events
  • Here you'll create, view and edit the Events for all of your Carpools. Events can be one or two-way, and can be created one at a time or as a repeating series. Each event has its own message thread, and tracks its changes so you'll always know who did what and when.

Create an Event

From the Main Menu go to Events and tap the "+" button on the upper right.

This new event is set as "Repeating" to show all options. Repeating events require both Start and End Date, as well as Repeat Every and Repeat Days values. The event name, location and address all come from the "Soccer" Carpool defaults, and can be changed if desired.

Date(s) and Location are required in this section.

Scroll down to the "To" section, outlined in blue. Blue always denotes a "To" driving direction in the app. Tap the "Select Time" button to expose the time picker and set the start time for the event. You can assign a driver here and remove riders as desired.

Time is always required if the direction is enabled.

Scroll down to the "From" section, outlined in orange. Orange always denotes a "From" driving direction. A driver is being assigned â€“ note that the driver selection also shows the number of riders the driver can take.

Time is always required if the direction is enabled.

Events can be one-way in either direction. To disable a direction, toggle the switch in either the "To" or "From" section. Here the "From" direction has been disabled.

Once all the required fields are provided, the "Save" button will activate and you can save the event.

Events List

After saving the repeating event an updated Events list will load. Event entries are grouped by date, and show the carpool name, event name, to/from time, assigned driver and rider count. Drives with an assigned driver also indicate how many open seats are available, with overfilled drives using a red badge.

The buttons at the bottom of the Events view are used to quickly filter the list. The car button will show only events where you are driving at least one direction. The checked person button will only show events where your kids are riding at least one direction. Use the calendar button to set how far into the future to display events.
