Event Details

From the event list, tap on any event to get to the event details view.

The detail view is used to assign yourself as a driver, edit event info like date, location or time, and to remove riders. The detail view can also be used to copy or delete an event and to read and add notes.

To un-assign yourself as Driver, swipe left on your name to expose the "Can't Drive" button.

Event changes are saved automatically and trigger the creation of a note so you always know when the event was changed and who made the change.

Tap on the "x Notes" button at the top right to show the note list. Use the button at the top right of the notes list to create a new note.
