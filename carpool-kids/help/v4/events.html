Events
  • Here you'll create, view and edit the Events for all of your Carpools. Events can be one or two-way, and can be created one at a time or as a repeating series. Each event has its own message thread, and tracks its changes so you'll always know who did what and when.

Create an Event

From the Main Menu go to Events and tap the "+" button on the upper right.

This new event is set as "Repeating" to show all options. Repeating events require both Start and End Date, as well as Repeat Every and Repeat Days values. The event name, location and address all come from the "Soccer" Carpool defaults, and can be changed if desired.

Date(s) and Location are required in this section.

Scroll down to the "To" section, outlined in blue. Blue always denotes a "To" driving direction in the app. Tap the "Select Time" button to expose the time picker and set the start time for the event. You can assign a driver here and remove riders as desired.

Time is always required if the direction is enabled.

Scroll down to the "From" section, outlined in orange. Orange always denotes a "From" driving direction. A driver is being assigned – note that the driver selection also shows the number of riders the driver can take.

Time is always required if the direction is enabled.

Events can be one-way in either direction. To disable a direction, toggle the switch in either the "To" or "From" section. Here the "From" direction has been disabled.

Once all the required fields are provided, the "Save" button will activate and you can save the event.

Events List

After saving the repeating event an updated Events list will load. Events are grouped by date, and show the Carpool Name, Event Name, To/From times, Location, assigned Driver and Rider count. An assigned driver row also has a green/gray/red badge showing the open seat count.

To filter the list to see only events where you are the assigned driver, tap the "My Driving Events" button in the bottom bar.

To filter the list to see only events where your kids are riding, tap the "My Kid's Events" button in the bottom bar.

To change how many months of events to display, tap the "Display Events For" button in the bottom bar

To view an animated chart showing how many drives (past & future) each driver has been assigned, tap the "Chart" button in the bottom bar. This is a Pro feature, but a demo chart is available as a preview.

If you've subscribed to Pro and have multiple carpools, use the "Filter" button to filter the list by carpool.

Edit an Event

Tap an event in the list to view the event in detail. Tap the time indicators to switch the drive direction – remember the "To" direction is always blue, while the "From" direction is orange. Use the buttons at the bottom to take the following actions:

Tap the "New Message" button to create a message. Messages are sent to all friends in the carpool, and are delivered by iOS push notification and/or e-mail, depending on each friend's notification settings.

Note: If "Suppress Notificiations" is enabled for the Carpool, a notification will not be sent.

Tap the "Message View" (speech bubble) on the upper right to view event Messages or History entries. Switch between Messages and History by tapping the switch in the list header.

Tap the "Edit Event" button if you want to change the event date, time, name, location or address.

If the event is repeating, you'll be given the option to apply the changes to "Event Only" or to "This & Future Events".

Note: A date change is only applied to the one event even if "This & Future" is selected.
