Create an Event

From the Main Menu go to Events and tap the "+" button on the upper right.

This new event is set as "Repeating" to show all options. Repeating events require both Start and End Date, as well as Repeat Every and Repeat Days values. The event name, location and address all come from the "Soccer" Carpool defaults, and can be changed if desired.

Date(s) and Location are required in this section.

Scroll down to the "To" section, outlined in blue. Blue always denotes a "To" driving direction in the app. Tap the "Select Time" button to expose the time picker and set the start time for the event. You can assign a driver here and remove riders as desired.

Time is always required if the direction is enabled.

Scroll down to the "From" section, outlined in orange. Orange always denotes a "From" driving direction. A driver is being assigned – note that the driver selection also shows the number of riders the driver can take.

Time is always required if the direction is enabled.

Events can be one-way in either direction. To disable a direction, toggle the switch in either the "To" or "From" section. Here the "From" direction has been disabled.

Once all the required fields are provided, the "Save" button will activate and you can save the event.

Events List

Saving will load an updated Events list. Events are grouped by date, and show the Carpool Name, Event Name, To/From times, Location, assigned Driver and Rider count. An assigned driver also has a green/gray/red badge showing their open seat count.

To filter the list to see only events where you are the assigned driver, tap the "My Driving Events" button in the bottom bar. Tap again to return to All Events.

To filter the list to see only events where your kids are riding, tap the "My Kid's Events" button in the bottom bar. Tap again to return to All Events.

To change how many months of events to display, tap the "Display Events For" button in the bottom bar

To view an animated chart showing how many drives (past & future) each driver has been assigned, tap the "Chart" button in the bottom bar. This is a Pro feature, but a demo chart is available as a preview.

If you've subscribed to Pro and have multiple carpools, use the "Filter" button to filter the list by carpool.

Event Detail

Tap an event in the list to view the event in detail. Tap the time indicators to switch the drive direction – remember the "To" direction is always blue, while the "From" direction is orange. Use the buttons at the bottom to take the following actions:

The "New Message" button creates an event message.

Messages are sent to all friends in the carpool, and are delivered by iOS push notification and/or e-mail, depending on each friend's notification settings.

Note: If "Suppress Notifications" is enabled for the Carpool, no notification is sent, but the message is still saved.

The "Message View" button (speech bubble) on the upper right is used to view event Messages or History entries. Move between Messages and History by tapping the switch in the list header.

The "Edit Event" button is used to change the event date, time, name, location or address.

If the event is repeating, you'll be given the option to apply the changes to "This Event Only" or to "This & Future Events".

Note: A date change is only applied to the one event even if "This & Future" is selected.

The "Duplicate" button uses the current event as a template to create a new event or repeating events.

To duplicate, provide a date for the new event. If you choose to make the duplicate "Repeating", you also must provide "Repeat Every" and "Repeat Days" values.

The "Map" button shows a map for the current active direction driving direction (To/From).

The map button will only be available if the event has an address. The map view contains a pin for the event address and the address of all riders who have a Default Location defined in their parent's Profile.

Tap a pin to see the address and access the "Driving Directions" button (car icon).

The "Delete" button removes the event. If the event is repeating, you'll be given the option to remove all future repeats (be careful!).
