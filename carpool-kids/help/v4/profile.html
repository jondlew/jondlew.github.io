Personal Info

Use the Profile to edit your personal information. Tap your e-mail address to change it or use the "Change Password" button if desired.

NOTE: If you created an account using Facebook or Google, only your phone number is editable.

Add a Kid

With "My Kids" selected, use the blue "+" button to add a Kid.

Once you've added a name, use the Save button. Then tap the placeholder image to add a picture for your Kid.

If you have other parents or friends that will be driving your Kid, you can add them as a "Linked Friend" once they have created an account and you've become friends. This will allow your Kid to be shared and avoid creating a duplicate Kid in multiple accounts.

Add a Location

Locations are where your Kids will be picked up and dropped off. They are not the places your carpool Events will take place. Tap Locations, then use the blue "+" to add a new one.

Only one Location can be the "Default", which is always used for your Kids' pickup/dropoff location unless overriden in the Event.

A Location must have a name and address before it can be saved. After naming the location, tap on "Lookup Address" and give the app permission to use your device location. This will allow the address lookup to search more accurately. Once an address is selected, the map view will update to show the location on the map. Don't forget to save!

Rider Seats

Other drivers in your carpool will want to know how many kids your car can hold. Use "Default Rider Seats" to let them know. You can always change this value in an Event if necessary. Tap the blue seat count number to pick a new value.
