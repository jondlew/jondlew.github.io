Event List

To view carpool events, tap on the "Events" button on the main menu.

The list view has three ways to manage which events are visible.

In its default state, the list view shows all events in chronological order, with events closest to today at the top.

Use the buttons at the bottom of the list to filter the events. The star button on the bottom left is used to show only events where you are assigned as a Driver.

The "Unfiltered" on the bottom right will only be visible if you've upgraded to a Pro account and are a member of more than one carpool. Use it to filter the list by specific carpool(s).

The events list can also be modified by setting the length of time used when pulling events from the cloud. To change the time period, use the settings button on the main menu (gear icon at lower left).

Tap on the blue "1 Month" text in the "Download Events For" row to expand a picker. There are options for 1, 3 or 6 Months, as well as 1 Year.
